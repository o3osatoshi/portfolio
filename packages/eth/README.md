# @repo/eth

ETH contract types and hooks generated by the Wagmi CLI.

## Environment setup

The `wagmi generate` workflow loads environment variables from
`packages/eth/.env.local`. The Doppler project for `portfolio-eth` is
temporarily removed, so secrets must be set locally for now.

1. Copy the template:

```bash
cp .env.local.example .env.local
```

2. Fill in the required values:
- `ETHERSCAN_API_KEY` â€” Etherscan API key used by `@wagmi/cli`.

## Commands

```bash
pnpm -C packages/eth generate
```

This will read `.env.local` and update `src/generated.ts`.
