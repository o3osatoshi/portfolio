# Documentation

This directory contains repository-level documentation.

## Sections

- [Conventions](./conventions/README.md): hand-written engineering conventions that apply across the monorepo.
- [Quality](./quality/README.md): testing and coverage references.
- [API](./api/README.md): generated API and type documentation.

## Notes

- `docs/conventions` is the canonical home for stable shared engineering rules across the monorepo, not package-specific usage guides.
- `docs/api` is generated documentation and should not be hand-edited.
- Package-level `README.md` files remain the primary place for package-specific behavior and setup.
